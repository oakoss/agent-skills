## Decision Tree

When the concept does not match the tables above:

1. Achievement/quality? -- Trophy, Star, Award, BadgeCheck
2. Money/cost? -- Tag, DollarSign, Wallet, Percent
3. Place/location? -- MapPin, Globe, Map, Navigation
4. People/users? -- User, Users, UserGroup
5. Communication? -- MessageCircle, Phone, Mail, Chat
6. Safety/security? -- Shield, Lock, ShieldCheck, Key
7. Time/speed? -- Zap, Clock, Timer, Calendar
8. Industry-specific? -- Check the Trade-Specific table
9. Generic positive? -- CheckCircle, ThumbsUp, Sparkles
10. Generic feature? -- Sparkles, Star, Zap
